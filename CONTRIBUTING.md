# Contributing

The main purpose of this repository is to continue to evolve the API, supporting more pieces, making it faster and easier to use.

Fell free to create issues and submit new Pull requests.

# Requirements

Node.js version 12.1.0 and npm version 6.9.0

# Installation

1 - Clone the repository using `https://github.com/caiodoneda/chess.git` or download the ZIP file.
2 - cd chess/
3 - run the command `npm install` to install all dependencies.

# Available Scripts

-   `npm run dev` to start the server with nodemon and have automatic restart when files change.
-   `npm run start` to start the server without hot reload.

Before starting the application, everything is compiled and outputted to the `dist` folder. `dist/index.js` is the application entry point.

In the dev environment, the server listens on port **5000** for connections
